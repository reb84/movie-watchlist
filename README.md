# 🎬 Movie Watchlist

A redesign of my original [Scrimba solo project](https://github.com/reb84/scrimba-frontend) movie watchlist app. I kept that repo untouched since part of the brief was to follow a Figma design file — so this is where I get to have fun with the styling.

This version is a deliberate trip back to the early web, built with a Neocities/Web 1.0 aesthetic in mind.


## Built with

- HTML
- CSS
- Vanilla JavaScript
- OMDb API
