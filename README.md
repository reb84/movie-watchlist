# Movie Watchlist

![Screenshot](imgs/screenshot.png)

✨ [LIVE DEMO](https://retro-movie-watchlist.vercel.app/)

A redesign of my original [Scrimba solo project](https://reb84.github.io/scrimba-frontend/12-movie-watchlist/) movie watchlist app. I kept that repo untouched since part of the brief was to follow a Figma design file — so this is where I get to have fun with the styling.

This version is a deliberate trip back to the early web, built with a Neocities/Web 1.0 aesthetic in mind.
